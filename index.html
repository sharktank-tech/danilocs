Projetos Recentes

Landing Page

Landing page simples, intuitiva e eficaz para os serviços de frete local. A página tem como principal objetivo fornecer informações essenciais sobre o serviço, como localização, horário de atendimento, e facilitar o contato com o cliente através de um botão que direcionará diretamente para o WhatsApp do prestador de serviço.

Ponto Eletrônico

Projeto próprio de uma aplicação desenvolvida para gerenciamento da marcação de ponto eletrônico. Funcionalidades: - Registro de novos usuários - Banco de dados dos usuários - Banco de dados das informações das marcações - Área de conta com informações sobre salário e carga horária - Envio de email com relatório. Tecnologias usadas: Python-Flask, SQL, HTML&CSS, Tailwind-CSS, JavaScript.

Gerenciador de Tarefas

Este é um projeto de gerenciamento de tarefas que permite adicionar, editar, excluir e visualizar tarefas. O projeto também inclui a funcionalidade de enviar um relatório por e-mail. Tecnologias usadas: Python-Flask, SQL, HTML&CSS, JavaScript.
